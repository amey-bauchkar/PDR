# Legacy Scripts Archive

**Status: DEPRECATED / NON-RUNTIME**

This directory contains historical migration and extraction utilities. These scripts were used to scrape and build the initial product catalogue, datasheets, and live data during the legacy transition phase.

They are preserved here purely for historical context or future reference in case data remapping is ever required. They are **not** part of the active development flow, and they are never executed during runtime or the Vite build process.
